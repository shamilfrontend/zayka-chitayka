# Зайка-Читайка

Игровое веб-приложение для детей 5–7 лет: учимся **читать и считать по-русски** — буквы, слоги, простые слова, цифры и числа.

Сайт: [https://zayka-chitayka.ru](https://zayka-chitayka.ru)

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно `http://localhost:5173/`).

## Сборка

```bash
npm run build
npm run preview
```

## Деплой на Beget

После `git push` в `main` GitHub Actions собирает проект и заливает `dist/` на Beget по SSH (rsync).

Нужные secrets в репозитории (Settings → Secrets and variables → Actions):

| Secret | Назначение |
|--------|------------|
| `SSH_PRIVATE_KEY` | Приватный ключ деплоя |
| `SSH_HOST` | Хост Beget |
| `SSH_PORT` | Порт SSH |
| `SSH_USER` | SSH-пользователь |
| `DEPLOY_PATH` | Путь к корню сайта на сервере |

Workflow: [`.github/workflows/deploy.yml`](.github/workflows/deploy.yml).

## Что внутри

- **PWA** — можно установить на домашний экран, работает офлайн после первого визита
- **Буквы / слоги / слова / цифры / числа** — карточки с озвучкой
- **Цифры 0–9** — запись, название и количество предметов
- **Числа 0–100** — запись и название; проверка из 20 случайных вопросов
- **Сложение / вычитание** — примеры по диапазонам; сдача раздела после 10 верных
- **Мини-игры и проверка раздела** — короткие сессии; после сдачи теста раздел отмечается в прогрессе
- **Звуки** — короткие тоны при верных и неверных ответах
- **Сброс** — кнопка «Сбросить прогресс» в настройках
- **Прогресс** — в `localStorage`

## Стек

Vue 3, TypeScript, Vite, Vue Router, CSS Modules, vite-plugin-pwa.
